# 阿里云 Zblog 框架案例

## 快速体验

- 初始化项目：`s init devsapp/start-zblog`
- 进入项目后部署：`s deploy`
- 部署过程中可能需要阿里云密钥的支持，部署完成之后会获得到临时域名可供测试
